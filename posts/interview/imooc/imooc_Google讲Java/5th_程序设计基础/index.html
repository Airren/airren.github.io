Java 拆箱与装箱

Primitive type vs Object

  • primitive type: int, long 值类型， 可以用a = b 去判断
  • Object: Integer, Long, Float, String 引用类型， a == b 判断是否为同一个Object， 用a.equals(b) 或者 Object.equals(a,b)][a 可以为 null]判断值是否相等。 Boxing and Unboxing
  • Integer a = 2; // Boxing
  • Integet b = new Integer(2); // Boxing
  • int v = a.intValue();
